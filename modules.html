Platform Modules

Driv‑en Version 1.0 includes four core modules designed to streamline field operations, safety, equipment management, and pipeline workflows.

Explore the Modules

Select a module below to learn more about its features and workflows.

  • Equipment Module
    Manage inspections, maintenance, readiness, documents, rentals, and GPS tracking.
  • Safety Module
    Conduct safety meetings, inspections, incident reporting, SDS, JSA/JHA, and compliance tracking.
  • Project Module
    Set up projects, manage reporting, documents, assignments, and field operations.
  • Pipeline Module
    Handle tally, weld map, coating, NDE, hydrotest, as‑builts, and pipeline construction workflows.
