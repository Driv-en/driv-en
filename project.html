Project Module

The Project Module centralizes project setup, reporting, documents, assignments, and field operations into one unified workflow.

Module Overview

The Project Module provides a complete operational view of every project — from setup and daily reporting to file storage, safety integration, equipment assignments, and closeout documentation.

Core Workflows

  • Project Setup: Client, location, scope, supervisors, crews, and project details.
  • Daily Reporting: Production, manpower, equipment usage, delays, weather, and notes.
  • Work Orders: Requests, approvals, tracking, and completion logs.
  • Change Orders: Scope changes, pricing, approvals, and documentation.
  • File Storage: Contracts, permits, drawings, photos, and project documents.
  • Safety Integration: Meetings, inspections, incidents, and project‑level safety logs.
  • Equipment Assignments: Equipment linked to projects with readiness and inspection status.
  • Project Requirements: Permits, certifications, client requirements, and compliance tracking.
  • Cost Codes & WBS: Work breakdown structure and cost code alignment.
  • Schedules: Milestones, timelines, and progress tracking.
  • Project Photos: Daily photos, progress documentation, and field uploads.
  • Closeout Package: Automatically compiled documents for client turnover.

Activity Log

Every action — reports, uploads, assignments, inspections, and changes — is logged automatically for full traceability and audit readiness.
